# Trung Tran Portfolio

Static portfolio template for Trung Tran's CV data.

## Stack

- Vite 8
- React 19
- TypeScript
- Tailwind CSS 4
- Framer Motion
- Radix UI primitives
- lucide-react icons
- class-variance-authority, clsx, and tailwind-merge for reusable interface components

## Scripts

```bash
npm install
npm run dev
npm run build
```

CV content lives in `src/data/cv.ts`. The current UI follows the editorial parchment style direction documented in `DESIGN.md`.
